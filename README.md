# 🎬 Netflix Data Analysis Using Python

A beginner-friendly data analysis project that explores Netflix's content library using Python. This project demonstrates the fundamentals of data analysis, including data cleaning, exploration, and visualization using the Netflix Titles dataset.

## 📌 Project Overview

The objective of this project is to analyze the Netflix dataset and gain insights into the platform's content by answering questions such as:

- How many Movies and TV Shows are available on Netflix?
- Which countries contribute the most content?
- How has Netflix's content grown over the years?
- What are the most common content ratings?
- Which genres are the most popular?

The project follows the basic data analysis workflow:
1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Data Analysis
5. Data Visualization

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/Netflix-Data-Analysis.git
```

2. Navigate to the project folder

```bash
cd Netflix-Data-Analysis
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Run the Python file

```bash
python netflix_analysis.py
```
